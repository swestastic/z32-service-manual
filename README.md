# Z32 Service Manual Android App

An Android application for viewing Z32 Nissan 300ZX service manual PDFs. There was an app on the Play store for a long time, but it has since been removed and no longer works on the newest versions of Android. This is its replacement!

I am not an Android developer, so there may be bugs, issues, etc. Please feel free to reach out if you encounter any problems or have any features you'd like added.

PDFs are sourced from NicoClub. I claim no ownership of these files, and only aim to make what is already publicly accessible easier to use for fellow Z32 owners.
